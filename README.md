## About

Write both languages in one file using `## EN` and `## RU` headings. This CLI exports separate snippets ready to paste.

## What's inside

- `scripts/` — CLI tool(s)
- `LICENSE` — MIT
- Bilingual README (RU below)

## Quick start

```bash
git clone https://github.com/YehhiAleksandra/bilingual-md.git
cd bilingual-md
python3 scripts/bilingual_md.py examples/post.md --lang EN
```

## Status

🚧 **v0.1** — small tool, maintained for daily content work.
